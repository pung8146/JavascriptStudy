.push(element) : 배열의 뒤에 엘리먼트 추가 .pop() : 배열의 뒤에서 엘리먼트 삭제하고 리턴 .shift() : 배열의 앞에서 엘리먼트 삭제하고 리턴 .unshift(element) : 배열의 앞에 엘리먼트 추가 var arr =[1,2,3,4,5]; arr.pop(); 5 --> arr [1,2,3,4] arr.shift(); 1 --> arr [2,3,4] arr.push(6); 4 --> arr [2,3,4,6] // 반한되는건 길이 arr.unshift(0); 5 --> arr[0,2,3,6]; // 반환되는건 길이 arr.reverse(); 값을 거꾸로 출력 arr.sort(); 다시 배열안의 값들을 정렬 배열에 concat(배열) 가능; 배열에 indefOf,lastIndexOf 사용가능; 문자열 split 함수 .문자열을 구분자(separator)로 나눠서 각각을 담은 배열을 반환하는 함수 .split()은 문자열을 분할하는 속성입니다. var str="1,2,3,4,5"; str.split(","); ["1","2","3","4","5"]
